# Commercial Licensing

This repository is available under a dual-license model:

- Community use defaults to `AGPL-3.0-or-later`.
- Separate commercial licenses are available from the copyright holder.

## When you need a commercial license

You need a separate commercial license if you want to do any of the following
without the copyleft obligations of `AGPL-3.0-or-later`:

- distribute the software as part of a closed-source product
- embed the software in a proprietary application or SDK
- run a modified version as a hosted service without releasing the
  corresponding source code required by the AGPL
- negotiate warranty, indemnity, support, or other contractual terms that are
  not part of the community license

## Scope

Any commercial license must be granted in a separate written agreement signed
by the copyright holder. This file is a licensing policy summary only. It does
not by itself grant any commercial rights.

Unless and until such an agreement exists, all source code use is governed
solely by `AGPL-3.0-or-later`.

## Brand assets

Commercial rights to use the `gpen` name, logos, or other brand assets are not
included automatically. Brand use requires separate permission unless the
written commercial agreement says otherwise.
