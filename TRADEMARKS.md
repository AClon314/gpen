# Trademarks and Brand Assets

The `gpen` name, logos, icons, product artwork, screenshots used for branding,
and other source-identifying assets are trademarks or brand assets of the
project owner unless a file explicitly says otherwise.

## What the code license does not grant

The open-source license for this repository covers source code. It does not
automatically grant the right to:

- use the `gpen` name for a modified fork in a way that implies it is the
  official project
- use the project logo, icon, or branded artwork for a fork or derivative
  product
- imply endorsement, partnership, certification, or origin from the original
  project owner

## Forks and redistributions

If you distribute a modified fork:

- you must remove or replace official logos, icons, and similar brand assets
  unless you have separate written permission
- you must not present the fork as the official `gpen` project
- you may describe your project factually as "based on gpen" or "a fork of
  gpen" as long as the statement is accurate and not misleading

## Commercial use

Commercial licensing of the code does not automatically include trademark
rights unless that permission is granted in the separate written commercial
agreement.
